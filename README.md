# Customer Churn Prediction

## Project Overview
This project predicts whether a telecom customer is likely to churn using Machine Learning.

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Random Forest

## Features
- Data preprocessing
- Label Encoding
- Random Forest Classifier
- Customer churn prediction
- Interactive Streamlit web application

## Model Accuracy
79.55%

## Dataset
IBM Telco Customer Churn Datasetcc
